In a standard document, elements follow a specific logical flow:

1. **The Heading (`<h1>`)**: Acts as the main subject of the page. It provides the largest text and the highest level of importance for search engines.
2. **The Paragraph (`<p>`)**: Provides the supporting detail. It is a block-level element that automatically adds vertical space to separate ideas.

* **Hierarchy**: By placing an `<h1>` above a `<p>`, you create a visual hierarchy that makes the page easy for users to scan.

* **Metadata Alignment**: The `<title>` in your `<head>` should generally match or relate closely to your `<h1>` content for the best user experience.
