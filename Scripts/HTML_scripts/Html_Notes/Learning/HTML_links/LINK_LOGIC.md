To create a working link, the browser requires specific components:

1. **The Anchor Tag (`<a>`)**: The container for the link.
2. **The HREF Attribute**: Short for "Hypertext Reference," this tells the browser the destination URL.
3. **Link Text**: The clickable content visible to the user.

When building links, keep these technical details in mind:

* **Absolute Paths**: Used for external sites (e.g., `https://google.com`).
* **Relative Paths**: Used for internal files (e.g., `contact.html`).
* **Target Attribute**: Adding `target="_blank"` will force the link to open in a new tab.
